# Frontend Design Skill

A Claude Code skill for creating distinctive, production-grade frontend interfaces with high design quality.

## Features

- **Design Thinking** — Bold aesthetic direction with intentional choices
- **Typography** — Distinctive, characterful font selections over generic defaults
- **Color & Theme** — Cohesive palettes with dominant colors and sharp accents
- **Spacing System** — 4pt/8pt grid for consistent rhythm
- **Responsive Design** — Mobile, tablet, and desktop support
- **Motion** — CSS animations, micro-interactions, scroll-triggered effects
- **Spatial Composition** — Asymmetry, overlap, grid-breaking layouts
- **Communication** — ASCII wireframes for layout, micro-syntax for animations

## Usage

The skill activates when you mention:
- "Build a landing page"
- "Create a dashboard"
- "Design a React component"
- "Style this UI"

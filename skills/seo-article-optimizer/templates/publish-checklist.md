# Publish Checklist (Single Article)

Use this checklist before publishing any optimized article.

## Required

- [ ] Unique title + H1 aligned (single H1, matches meta title intent)
- [ ] Meta title (50-60 chars, includes primary keyword + brand)
- [ ] Meta description (150-160 chars, includes keyword + CTA)
- [ ] Slug finalized (short, readable, hyphenated, keyword-rich)
- [ ] At least 3 internal links to key pages
  - [ ] Early link (first 20% of content)
  - [ ] Mid-content link
  - [ ] CTA/conclusion link
- [ ] All images have descriptive alt text
- [ ] One clear CTA aligned to site goal
- [ ] Heading hierarchy validated (H1 → H2 → H3, no skipped levels)

## E-E-A-T Signals

- [ ] Author name and bio visible
- [ ] Publication/update date displayed
- [ ] Data sources cited with links
- [ ] Original insights or first-hand experience included

## Snippet Opportunities

- [ ] Definition paragraph (40-60 words) if applicable
- [ ] Numbered steps or bullet list if applicable
- [ ] Comparison table if applicable
- [ ] FAQ section with 3-6 questions if applicable

## Technical

- [ ] Page loads in < 3 seconds
- [ ] Mobile-friendly layout verified
- [ ] No broken links
- [ ] Canonical URL set correctly

## Post-Publish

- [ ] Request indexing via Google Search Console
- [ ] Share on social channels
- [ ] Monitor initial rankings (7-14 days)
- [ ] Check for indexation (site:example.com/slug)
